How to Get Your API Key

Step 1: Go to platform.openai.com and log in to your Chat GPT Account.

Step 2: Once you've logged in, locate the icon displaying your account name in the top right corner of your screen. Simply click on it to access the dropdown menu, and from there, select "View API keys."

Step 3: Now that you're in the API keys section, you should find a button labeled "Generate new secret key." Click on this button to create a fresh API key.

Step 4: A box will appear, prompting you to assign a name to your secret API key. It's advisable to use distinct names for different applications or websites to ensure you can easily identify their purpose in the future. Once you've provided a name for your key, proceed by clicking the "Create secret key" button.

Step 5: Finally, you'll be presented with the generated secret API key. Ensure you copy this secret key and paste it into the application where it's required. It's very important that you copy and securely store this key since it cannot be retrieved for security purposes. This key is essential for authenticating your applications with our services.

Step 6: Go to cntrl.ai and log into your account. Click on “Settings” at the top of your Cntrl AI dashboard.

Step 7: After clicking the “Settings” tab you will see the system settings box. In the System Settings, you will see “Open API Key.” This is where you will place your API key that you have generated. Then make sure to click “Save All Settings” at the bottom to save your API key.
